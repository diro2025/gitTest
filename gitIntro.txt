Quick Git Tutorial:

$git init -- initializing git
$git clone <url> -- get entire repo codebase from URL
$git pull  --pull from repo to make sure you have current codebase
$git status  --get current status for git
$git add <file name>  --add or update file before commit
$git commit -m "description message"
$git push  -- push commit file to git repo
$git rm <filename> --remove file 
$git mv <current path> <new path> ---change file path
